How to set up Dropbox storage (5 minutes)

  1. Go to dropbox.com/developers/apps and click Create app.
  2. Choose Scoped access → App folder → name it ARA2026.
  3. Under Permissions, enable files.content.write and files.content.read.
  4. Under Settings, scroll to OAuth 2 → Generated access token and click Generate.
  5. Copy the token. Go to the app URL and paste it in the setup banner at the top.
  6. Each participant submission is saved as a JSON file in Apps/ARA2026/ in your Dropbox.
  7. You see all submissions here in the admin panel. Participants only see their own matches.

How to host on GitHub Pages (free)

  1. Create a new GitHub repository (e.g. ara2026-matching).
  2. Upload ARA_Collaboration_App.html and rename it to index.html.
  3. Go to Settings → Pages → Branch: main and click Save.
  4. Your app is live at https://yourusername.github.io/ara2026-matching in ~2 minutes.
  5. Generate a QR code for that URL (any free QR generator) and add it to slide 8.
  6. The token is baked into the HTML file — participants never see it or need to configure anything.

Privacy model: Participants submit → their data goes directly to your Dropbox App folder. They never see each other's data. The matching algorithm runs in their browser using the speaker list + aggregate keyword pool. You are the only one who can see the full submission list.
